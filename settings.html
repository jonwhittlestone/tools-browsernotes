Browser Notes Settings

Enables Vim-style keyboard navigation in the notes editor

Vim Mode Commands

Normal Mode

  • i - Enter insert mode
  • a - Append after cursor
  • o - Open new line below
  • O - Open new line above
  • v - Visual mode
  • V - Visual line mode

Navigation

  • h - Move left
  • j - Move down
  • k - Move up
  • l - Move right
  • w - Next word
  • b - Previous word
  • 0 - Start of line
  • $ - End of line
  • gg - Go to top
  • G - Go to bottom

Editing

  • x - Delete character
  • dd - Delete line
  • yy - Yank (copy) line
  • p - Paste after
  • P - Paste before
  • u - Undo
  • Ctrl+r - Redo

Insert Mode

  • Esc - Return to normal mode
  • Ctrl+[ - Return to normal mode
